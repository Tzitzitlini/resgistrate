3.-Enviar Datos

Investigación

Form: Es un conjuto de controles(botones,cajas de texto, casillas de verificación, botones, radio, etc)que permiten al usuario introducir datos y enviarlos al servidor web para su procesamineto. Es un elemento de bloque, en su interior puede haber cualquier elemento típico de una página web.

Fieldset:Permite agrupar un conjunto de controles, los navegadores muestran una caja alrededor de cada grupo de controles

Legend:Permite añadir una leyenda al fieldset, los navegadores muestran la leyenda sobre el borde que rodea al grupo de controles.

Strong: Esta definido como elemento de frase, crea una caja en línea, es el apropiado para marcar con especial énfasis las partes más importantes de un texto

Input type="": Es una caja de texto de una sola línea, mientras que para type existen varios atributos entre ellos, text, submit, number y radio, solo por mencionr algunos.Existen también cajas de texto de contraseña que permite que no sea visible lo que se escribe porque aparecen puntos negros.

textarea: Área de texto, caja de texto de varias líneas, sus atributos obligatorios rows y cols establecen el número de filas y columnas iniciales de la caja, aunque los navegadores permiten modificarlo arrastrando la esquina inferior derecha.
